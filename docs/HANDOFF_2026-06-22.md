## What ran (results)
**Phase 2 — QLoRA (4× H100 DDP):** Qwen2.5-3B, 4-bit NF4, LoRA r=16, 1 epoch, **1,251,225 benign** user-days, eff batch 32. `train_loss=0.143` (converged ~0.13 early, then plateaued). Adapter: `checkpoints/qwen3b_session_qlora_ddp/adapter/` (114 MB).

**Phase 3 — Delta extraction (layers 12/18/24, mean-pooled):** 142,072 eval examples (70 positives). `base_nll 2.70 → adapted 0.44` (adapter strongly fit the sessions). `delta_nll` benign **−2.262** vs malicious **−2.208** (malicious improved *slightly less* → faint residual signal, consistent with benign-only training).

**Phase 4 — Delta-SAE frontier (36 configs):** best top5−control3 selectivity = **Layer 18** (mult2/k8 → **0.614**; mult4/k4 → 0.605) > L24 (~0.53) > L12 (~0.49). Sparser/smaller dictionaries = more selective; controls ~0. **PROXY selectivity (removed-reconstruction-energy inside the SAE), NOT causal patching.**

## Key decisions & learnings (non-obvious)
- **1 epoch was overkill** — loss plateaued early on this low-entropy structured data. Next runs: early-stop / fewer steps (real SU savings).
- **Eval-cost trap:** full-val eval every 500 steps ≈ +20 h → capped `MAX_VAL=2000` (training-time val loss is only a monitor; real eval is Phase 5).
- **Extraction OOM:** batch 32 OOM'd (Qwen **152k-vocab fp32 logits** × long eval sequences). **batch 8** is the safe ceiling at seq 2048; that's what produced the final deltas.
- **Fast JSONL builder** (`scripts/build_session_jsonl_fast.py`) replaced the ~2 h slow builder — **~251×, byte-identical** (validated).
- **Latent data bug to decide on:** the original `build_session_jsonl.py` silently **drops 4 hyphen-named session cols** (`file_n-to_usb1`, `file_n-from_usb1`, `file_n-file_act3`, `file_n-disk1`) via pandas `itertuples` rename. The fast builder reproduces this for fidelity. **Decide whether to include those 4 cols** (changes the training representation).
- DDP validated against current best practices: `device_map={"":local_rank}` (not "auto"), `ddp_find_unused_parameters=False` set explicitly (Trainer default is True for a PeftModel!), gradient-checkpointing `use_reentrant=False`.

## The decisive open question
Proxy selectivity ≠ patchability. **Not yet answered:** does the delta-SAE beat the session-AE on **sufficiency/patchability**? That requires Phase 5.
